# 🚘 AutoCatalog+

**AutoCatalog+** — это десктопное приложение на Python, предназначенное для просмотра каталога автомобилей.  
Приложение позволяет пользователям изучать различные автомобильные бренды и модели, просматривать технические характеристики и ориентировочную стоимость автомобилей, а также сохранять интересующие модели в избранное.

Проект разработан как приложение для **любителей автомобилей**, которые хотят удобно изучать информацию о разных моделях машин в одном месте.

---

# 📌 Основной функционал

### 🚗 Каталог автомобилей
- просмотр списка автомобильных брендов  
- просмотр моделей выбранного бренда  
- переход на страницу модели автомобиля  

### 📊 Характеристики автомобилей
Каждая модель автомобиля содержит подробную информацию:
- тип кузова  
- годы выпуска  
- объём двигателя  
- мощность двигателя  
- тип топлива  
- тип привода  
- тип коробки передач  
- разгон 0–100 км/ч  
- максимальная скорость  

### 💰 Средняя цена автомобиля
Для каждой модели отображается **средняя ориентировочная цена**, что помогает пользователям сравнивать автомобили.

### 🔎 Поиск и фильтрация
Пользователь может искать автомобили по:
- названию бренда  
- названию модели  
- типу кузова  
- типу топлива  
- диапазону цены  

### 👤 Пользовательские аккаунты
В приложении реализована система пользователей:
- регистрация нового аккаунта  
- вход в систему  
- хранение данных пользователя  

### ⭐ Избранные автомобили
Авторизованные пользователи могут:
- добавлять автомобили в избранное  
- удалять их из избранного  
- просматривать список сохранённых автомобилей

# 🖥 Интерфейс приложения

Приложение содержит несколько основных экранов:

### Главный экран
- список автомобильных брендов  
- строка поиска  
- навигация по приложению  

### Экран бренда
- информация о бренде  
- список доступных моделей  

### Экран модели
- фото автомобиля  
- технические характеристики  
- средняя цена  
- кнопка добавления в избранное  

### Окно авторизации
- вход в аккаунт  
- регистрация  

### Профиль пользователя
- информация о пользователе  
- список избранных автомобилей
