# AutoCatalog

AutoCatalog — это десктопное приложение на Python для просмотра каталога автомобилей с их техническими характеристиками и средней рыночной ценой.

Приложение позволяет пользователям изучать различные автомобильные бренды и модели, узнавать их основные параметры и сохранять интересующие автомобили в избранное.

## Возможности

- просмотр брендов автомобилей  
- просмотр моделей каждого бренда  
- технические характеристики автомобилей  
- отображение средней цены  
- поиск и фильтрация  
- регистрация и вход пользователей  
- добавление автомобилей в избраное

# Цель проэкта

Проект создан для любителей автомобилей, которые хотят удобно изучать различные модели машин, их характеристики и ориентировочную стоимость в одном приложении.
